# ✨ Konsep UI/UX (User Experience)

## 🎨 Tema Visual: "Premium Forest & Gold"
Terlihat dari palet warna yang sudah ada:
*   **Forest Green**: Memberikan kesan aman, terpercaya, dan organik (lingkungan).
*   **Gold/Amber**: Untuk aksen premium dan highlight status prioritas.
*   **Soft Paper/Cream**: Sebagai latar belakang untuk mengurangi kelelahan mata (*eye strain*).

## 🗺️ User Journey (Flow)
1.  **Dashboard**: Overview statistik total penduduk dan ringkasan kelayakan.
2.  **Data Entry**: Input data penduduk & titik koordinat melalui peta (GIS).
3.  **Survey Data**: Input kondisi fisik rumah berdasarkan 4 Pilar (Keselamatan, Kesehatan, Luas, Komponen).
4.  **Automated Scoring**: Sistem menghitung secara *real-time* menggunakan Fuzzy Mamdani.
5.  **Visualization**: Melihat persebaran rumah di peta untuk menentukan prioritas bantuan.

## 💡 UX Principles
1.  **Feedback Instan**: Memberikan notifikasi sukses/gagal yang jelas saat menyimpan data.
2.  **Data Visualization**: Daripada hanya tabel, gunakan Chart dan Map untuk memudahkan pengambilan keputusan oleh Camat.
3.  **Zero-Configuration**: Admin tidak perlu pusing memikirkan rumus fuzzy; cukup input data mentah, sistem memberikan hasil jadi.
4.  **Consistency**: Font Serif untuk judul (kesan elegan/formal) dan Sans-serif untuk data (keterbacaan).
