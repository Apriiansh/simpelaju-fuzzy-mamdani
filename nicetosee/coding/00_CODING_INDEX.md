# 🛠️ SIMPELAJU — Coding Tutorial
> Panduan praktis membangun sistem SPK RTLH menggunakan Laravel & Fuzzy Logic.

Tutorial ini akan membimbing Anda melalui langkah-langkah teknis untuk membangun aplikasi Simpelaju dari sisi pengkodean. Pastikan Anda sudah menginstal PHP, Composer, dan MySQL.

---

## 🚀 Alur Pengembangan (Development Flow)

```text
STEP 1: DATA LAYER
01_DATABASE_MIGRATION_DAN_SEEDER.md
└─ Membuat tabel & data awal

STEP 2: MODELS & RELATIONS
02_MODEL_DAN_RELASI_ELOQUENT.md
└─ Menghubungkan antar tabel data

STEP 3: THE BRAIN (FUZZY)
03_IMPLEMENTASI_FUZZY_ENGINE_PHP.md
└─ Coding logika matematika Mamdani

STEP 4: BACKEND LOGIC
04_CONTROLLER_DAN_VALIDASI_FORM.md
└─ Menangani input & trigger fuzzy

STEP 5: FRONTEND UI
05_FRONTEND_DENGAN_TAILWIND_BLADE.md
└─ Desain Dashboard & Form

STEP 6: GIS INTEGRATION
06_INTEGRASI_LEAFLET_JS.md
└─ Memasang peta interaktif
```
